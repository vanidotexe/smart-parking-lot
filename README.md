A progressive web application for the city of Iași that helps drivers find free parking spots. The front end is built with React (TypeScript), the back end is a Spring Boot REST API, and a Raspberry Pi module uses a camera to detect which spots are occupied and publish data to the system.

**Features**
- **Real‑time availability:** Shows free and occupied parking spots across the city.
- **Computer vision detection:** Raspberry Pi camera module determines spot occupancy.
- **Reservations:** Reserve a spot and manage current reservations.
- **Favorites:** Mark preferred parking lots for quick access.
- **Admin tools:** Manage parking lots, spots, users, and notifications.
- **Payments:** Pay for reservations within the app via Stripe.
- **Support:** Built‑in helpdesk and live chat.
